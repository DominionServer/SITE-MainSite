Project Nebula Stress Test

Saturday 9th May 2020

When and What

This is a group event to stress test our server

In order to see if the Dominion server can handle 40+ players at once we’re performing a stress test event. We’d like to invite you all to join however please be aware that this event may lag, it may not… that’s the whole point! The point of this event is to figure out if we can host these large scale events so that in future we can host these on a more regular basis.

This event will be on Saturday 9th May 2020 at 6pm BST see this link for your time zone. It will be a flower power UHC and is designed to be a shorter game so that we may test the server itself and although hosted on Dominion will be run by Nebula.

What is UHC?

UHC stands for ultra hardcore.

It’s a Minecraft game mode where everyone starts off scattered randomly in the world with nothing. They have to gear up, battle monsters, and fight players to be the last man/team standing. The catch is: you cannot regenerate health naturally. You must use a golden apple or a healing potion. More information can be found here.

This event will be a team of 2 game. That means you get to choose your teammate! But please note that you do not need a chosen teammate to participate; if you don’t have a teammate you will be randomly paired with someone. For this reason, Discord and a working microphone are REQUIRED. UHC is a fast-paced gamemode, so typing to communicate just doesn’t work.

Settings

These are all the most important settings for this event

  1. Map size is 2000x2000
  2. Shrinks to 200x200 starts at PvP time until meetup time
  3. PvP 10 minutes in
  4. Meetup 30 minutes in
  5. Golden heads are on but only heal 4 hearts
  6. Notch Apples are on
  7. Absorption is on
  8. Ender Pearl Damage is off
  9. Apple drop rate is 20% and sheers may be used to drop apples. All leaf types drop apples.
  10. Flint rates set to 50%
  11. Nether is off
  12. No skybasing during meetup

Scenario Information

These spice up the UHC games and allow us to keep things a little fresh.
  • Hastey Boys: All tools have efficiency 3 and unbreaking 1.
  • Flower Power: When a player breaks any random flower, the flower will drop a random item.

What are the rules?

These are the rules for any UHC run by Dominion or if we are invited to another server.
  1. Toxicity is NOT ALLOWED. We understand that you may get heated and it’s fine to have a little banter however toxicity which gets out of hand will not be aloud and we’ll ensure you know when it is.
  2. Use of mods outside of Badlion client and performance mods (such as optifine) are NOT ALLOWED at this event. You are also not allowed to use toggle sneak within Badlion as this allows you to sneak whilst in your inventory.
  3. Xray or cave finders are NOT ALLOWED at this event. This should go without saying but do not install X-ray and expect us not to find out and remove you. Use of X-ray could lead to you being banned from the respective server you are apart of.

How to apply

In the Dominion Discord Events channel is a post on this event, if you emote on that event you'll be notified within 24 hours of the event with instructions. This will tell you how to join the server and how to get the correct role. Unless you are planning on playing the Nebula server there is no reason to join before this time.

How to prepare

On Wednesday we will run our UHC event however this time we will run the event with Minecraft 1.8.9. It will not have all the settings and Scenarios but it will at least prepare you for the change in combat and recipes and the like for the event on Saturday. This is also open to any non-whitelisted member who wishes to join and teaming up is also allowed in groups of two.
